1. Local Transactions:
-------------------
In a local transaction, all operations of a transaction are executed in the same database.

example:
--------
Funds transfer from one account to another account where both accounts belong to the same bank.

2. Global Transaction:
---------------------
In global transaction, the operations of a transaction are executed on different databases.

example:
--------
Funds transfer from one account to another account where both accounts belong to different banks.

ACID RULES:
>ATOMICITY(A):-ALL (OR) NOTHINNG.
>CONSISTENCY(C) :- THE DATABASE REMAINS IN A VALID STATE.
>ISOLATION(I) :-TRANSACTION DO NOT INTERFACE WITH EACH OTHER.
>DURABILITY(D) :-COMMITED CHANGES ARE PERMINENT.
